# brandonkramer/homebrew-tap

Homebrew formulas for my tools.

## Install

```bash
brew install brandonkramer/tap/trelly
brew install brandonkramer/tap/netcup
```

## Formulas

| Formula | Description |
|---------|-------------|
| [trelly](Formula/trelly.rb) | Fast Trello CLI + MCP server — multi-profile auth, kanban TUI · [repo](https://github.com/brandonkramer/trelly) · [npm](https://www.npmjs.com/package/trelly) |
| [netcup](Formula/netcup.rb) | CLI and ops TUI for the netcup SCP REST API · [repo](https://github.com/brandonkramer/netcup-cli) |

`trelly` is bumped automatically by [its release workflow](https://github.com/brandonkramer/trelly/blob/main/.github/workflows/release.yml). `netcup` is bumped from [GitHub Releases](https://github.com/brandonkramer/netcup-cli/releases) after each tagged GoReleaser publish.
